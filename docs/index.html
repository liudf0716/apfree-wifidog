ApFree WiFiDog

下一代 WiFi Portal 网关 · eBPF 加速与 DPI 流控

GitHub 功能亮点 快速上手

核心功能 Features

认证系统

支持 HTTP/HTTPS 重定向、多用户认证与外部 Auth Server 通讯。

eBPF 流控 & DPI

内置 eBPF 加速通道，支持协议识别与带宽控制。

系统架构 Architecture

快速上手 Quick Start

opkg install apfree-wifidog.ipk
/etc/init.d/wifidogx start

可通过 UCI 配置参数化部署到 OpenWrt LuCI 界面。
