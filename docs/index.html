ğŸ“– Introduction

ApFree WiFiDog is an open-source, high-performance captive portal for HTTP and HTTPS, tailored for the OpenWrt platform.

It provides secure authentication, high concurrency handling, and flexible rule management for Wi-Fi networks.

ğŸš€ Features

Stable & Fast

API-based iptables integration, thread-safe. Built with libevent2 + epoll for high performance.

Secure & Real-time

Full HTTPS redirection support and long connection support (WebSocket, MQTT).

Flexible & Dynamic

Local + Cloud authentication, splash page mode. Manage MAC, IP, domains without restart.

eBPF Support

Traffic control & DPI via eBPF.
